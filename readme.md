# Math.round()
### Метод Math.round() округляет число до ближайшего целого числа. 2,49 будет округлено в меньшую сторону (2), а 2,5 — в большую сторону (3).
# Math.ceil()
### В JavaScript метод Math. Функция ceil() возвращает наименьшее целое число, большее или равное заданному числу. Например, если у вас есть число 3,14, Math. ceil(3.14) вернет 4 .
# Math.floor()
### Floor() — это метод JavaScript, который возвращает наибольшее целое число, меньшее или равное заданному числу. По сути, он округляет число до ближайшего целого числа.
